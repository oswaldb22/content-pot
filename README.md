# Content Pot

Add consumable content to your pot.
Easily bookmarks/save for later content.

Content are stored in your browser's local storage.

Everything is free and open source. No credit required.


## Installation

```bash
pnpm install
```

```bash
pnpm run dev
```

```bash
pnpm run build
```


## Features

- Add consumable content to your pot.
- Easily bookmarks/save for later content.
